# Indigo Protocol

DeFi skills for the Indigo Protocol on Cardano.

## Prerequisites

- Node.js 20+
- `@indigoprotocol/indigo-mcp` package

## MCP Server

```bash
npx @indigoprotocol/indigo-mcp
```

## Tools

- CDP management (open, close, deposit, withdraw, mint, burn)
- Stability pool operations (create, adjust, close accounts)
- Staking (open, adjust, close positions)
- Price feeds and protocol stats
- DEX integration and swaps
