# Stability Pool Queries

Query stability pool state and accounts.

## Tools

### get_stability_pools
List all stability pools with their total deposits, earned rewards, and iAsset type.

### get_stability_pool_accounts
List all accounts in a specific stability pool with their deposit amounts and pending rewards.

### get_sp_account_by_owner
Look up a stability pool account by owner address. Returns deposit amount, rewards, and pending requests.
