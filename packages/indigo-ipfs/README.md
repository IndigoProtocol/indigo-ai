# @indigoprotocol/indigo-ipfs

IPFS and Collector skill package for Indigo Protocol on Cardano.

## Tools

| Tool | Description |
|------|-------------|
| `store_on_ipfs` | Store text content on IPFS |
| `retrieve_from_ipfs` | Retrieve content from IPFS by CID |
| `get_collector_utxos` | Get collector UTXOs for fee distribution |

## Sub-Skills

| Sub-Skill | Description |
|-----------|-------------|
| [IPFS Storage](sub-skills/ipfs-storage.md) | Store and retrieve data on IPFS |
| [Collector](sub-skills/collector.md) | Query collector UTXOs for fee distribution |

## Installation

```bash
npm install @indigoprotocol/indigo-ipfs
```
