# @indigoprotocol/indigo-cdp

CDP & Loan Management skill for the Indigo Protocol on Cardano.

## MCP Tools

| Tool | Description |
|------|-------------|
| `open_cdp` | Open a new CDP |
| `deposit_cdp` | Deposit collateral |
| `withdraw_cdp` | Withdraw collateral |
| `close_cdp` | Close a CDP |
| `mint_cdp` | Mint iAssets |
| `burn_cdp` | Burn iAssets |
| `analyze_cdp_health` | Check CDP health |
| `liquidate_cdp` | Liquidate unhealthy CDP |
| `redeem_cdp` | Redeem against CDP |
| `freeze_cdp` | Freeze a CDP |
| `merge_cdps` | Merge CDPs |
| `leverage_cdp` | Leveraged CDP via LRP |
| `get_all_cdps` | List all CDPs |
| `get_cdps_by_owner` | CDPs by owner |
| `get_cdps_by_address` | CDPs by address |

## Sub-skills

- **CDP Basics** — open, deposit, withdraw, close
- **Mint & Burn** — mint and burn iAssets
- **CDP Health** — analyze health and liquidation risk
- **Liquidation** — liquidate, redeem, freeze, merge
- **Leverage** — leveraged CDP via LRP
