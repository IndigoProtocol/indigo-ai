# Token Prices

Query current prices for ADA and INDY tokens.

## MCP Tools

| Tool | Description | Parameters |
|------|-------------|------------|
| `get_ada_price` | Get the current ADA price in USD | None |
| `get_indy_price` | Get the current INDY token price in ADA and USD | None |

## Examples

### Get ADA price

Use `get_ada_price` to retrieve the current Cardano ADA price in USD.

### Get INDY price

Use `get_indy_price` to retrieve the current INDY governance token price in both ADA and USD.
